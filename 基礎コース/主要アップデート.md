# 基盤技術
## AWS Graviton4
汎⽤コンピューティング向けカスタムシリコンの最新モデル
Amazon EC2 R8gに同チップを搭載する

## AWS Trainium2
機械学習ワークロード向けカスタムシリコンの第2世代
Amazon EC2 Trn2に同チップを搭載する
処理能力向上だけでなく、エネルギー消費効率も上がる

## Amazon S3 Express One Zone
⾼いパフォーマンスのデータアクセスを提供する新しいストレージクラス
アクセススピード向上だけでなく、リクエスト費用も下がる

## Grover
Amazon Auroraの処理を行っているもの

## Amazon Aurora Limitless Database
秒間数百万の書き込み、ペタバイトのデータ規模までスケールするデータベース
データとクエリを⾃動的に分散し、シャーディングの開発・運⽤が不要

# 生成系AI
## Amazon Q
新しいタイプの⽣成系AIによるアシスタント
AWSコンソール上でチャットで質問できる
VSCODEでAMAZON Qのチャットを使えば質問できる

## Amazon Bedrockのアップデート
Agent コーディングレスで開発できる
Knoledge Base 検索結果を要約する
Guardrails 不適切な回答をしない機能

## PartyRock
楽しく、直感的に⽣成系AIアプリケーションを構築できる「遊び場」

## Amazon DataZone AI recommendations
⽣成系AIによりデータカタログにビジネスコンテキストを⾃動的に追加

## myApplications in the AWS Management Console
アプリケーションを登録するとコスト、パフォーマンス、セキュリティなど主要なデータを⼀括表⽰し、アプリケーションの最適化を促進

## Amazon CloudWatch Application Signals
AWSで稼働するアプリケーションの情報を⾃動的に収集し可視化
サービスレベル⽬標（SLO）を定義して健全性をモニタリング

## Amazon Inspector CI/CD Container Scanning
CI/CDツールからAmazon Inspectorによるコンテナイメージスキャンが可能
